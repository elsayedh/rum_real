Chart of Accounts Hierarchy

CodersFort

Chart of accounts hierarchy defines how accounts are related to each other, This module will adds the parent id of each Account and bulid tree structure relation between Accounts Visually.

Installation

Copy accounts_hierarchy module to addons folder

Install the module normally like other modules.

Configuration

Enable Show Full Accounting Features and Show Chart of Account Hierarchy



Now you can see the Chart of Accounts Hierarchy menu Under Accounting.



Open the Chart of Accounts menu under Accounting configuration and Update Parent Account for each Accounts
